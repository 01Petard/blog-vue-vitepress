# 薪资对比

## 一、核算规则、公式

- 个人公积金 = 公积金基数 × 公积金比例，公司公积金与个人等额
- 社保个人缴纳：基数 × 10.5%（其中养老8%、医疗2%、失业0.5%）
- 月到手 = 月薪 - 个人社保 - 个人公积金，不考虑个人所得税
- 月总收益 = 月到手 + 个人公积金 + 公司公积金 = 月薪 - 个人社保 + 公司公积金
- 年包 = 月总收益 × 12，仅考虑12薪固定薪资，不考虑年终奖、项目奖金等其他收入
- 时薪根据年收益、公司工作每月出勤天数和工作时长进行综合计算，可作为工作性价比参考值
