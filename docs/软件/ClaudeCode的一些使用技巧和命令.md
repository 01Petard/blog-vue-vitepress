# Claude Code：终端AI开发助手

**核心理念**
只需说明你想做什么，剩下的编码、配置、调试等工作，由 Claude Code 自动完成。它不是普通的聊天机器人，而是一个能直接操作你电脑、执行完整开发流程的智能助手。

------

## 主要功能

- 自动创建或修改项目文件  
- 在终端中运行命令（如编译、测试、部署）  
- 自动安装依赖、排查错误、启动服务  
- 记住项目上下文，支持长时间连续开发对话

------

## 它带来的改变

- **大幅提升开发效率**：重复性工作几乎不再需要手动处理  
- **开发者重心转移**：从写代码转向设计系统结构和业务逻辑  
- **开启新开发范式**：让 AI 成为你日常开发流程的一部分

------

## 三种使用模式（按 Shift + Tab 切换）

1. **默认模式（Default）**
   每次修改前都会询问确认，适合正式项目或生产环境。
2. **自动模式（Auto）**
   全程自动执行，无需打断，适合快速原型或本地开发。
3. **规划模式（Plan）**
   只输出开发步骤和建议，不实际操作文件，适合前期架构讨论。

------

## 常用命令速查

| 功能               | 命令                             |
| ------------------ | -------------------------------- |
| 启动助手           | `claude`（在项目目录中运行）     |
| 恢复上次会话       | `claude -c` 或输入 `/resume`     |
| 查看当前上下文     | `/context`                       |
| 精简上下文（推荐） | `/compact`                       |
| 清空上下文         | `/clear`                         |
| 查看待办任务       | `/tasks`                         |
| 终止当前任务       | 按 `K`                           |
| 初始化项目规范     | `/init`（生成 `CLAUDE.md` 文件） |
| 切换到规划模式     | `/plan`                          |
| 切换到自动模式     | `/auto`                          |

------

## 推荐使用流程

1. 进入你的项目目录：`cd your-project`  
2. 启动助手：`claude`  
3. 初始化项目规范：`/init`  
4. 先用 **Plan 模式** 梳理整体架构  
5. 再切到 **Auto 模式** 让它自动实现  
6. 开发过程中定期用 `/compact` 保持上下文清晰高效
