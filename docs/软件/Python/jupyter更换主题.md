# 安装

```shell
pip install jupyterthemes
```

# 查看可选择的主题

```shell
jt -l
```

# 安装主题

```shell
jt -t 主题名 -T -N![](https://cdn.jsdelivr.net/gh/01Petard/imageURL@main/img/images)
```
